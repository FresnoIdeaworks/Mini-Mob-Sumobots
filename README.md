# Mini-Mob-Sumobots
This collection of code includes the Fresno Ideaworks 3rd generation sumobot, based on the MiniMob robot originally produced by the Citizen Science Workshop (https://www.tindie.com/stores/CitizenScientificWorkshop/).
While their online store appears to have not been active for several years, their robot kits do show up from time to time (our makerspace was gifted over a hundred of the kits a few years back).
The code in this repository is primarily the original MiniMob Ardublockly code available circa 2016.
We have included the original MiniMob code along with the Fresno Ideaworks SumoBot modifications for MiniMob SumoBot competitions.  You will also need to download the Ardublockly graphical environment from Carlos Pereira Atencio's repository at https://github.com/carlosperate/ardublockly/releases/.  Choose the release to matches your home computer environment (Mac/Windows/Linux), download it and unpack the zip file.  
You will also need to download and install the most recent version of the Arduino IDE (Integrated Development Environment) which can generally be found at https://www.arduino.cc/en/software/#ide.  There is an online version of the Arduino environment, but it is not clear that it will work with Ardublockly.  
Once the Arduino IDE is installed, you can run the Ardublockly application and will need to configure it to locate the Arduino IDE ("Arduino IDE.exe" in Windows).
